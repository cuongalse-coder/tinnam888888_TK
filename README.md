# 📋 DocCompare - So sánh Chứng từ XNK

Ứng dụng web so sánh chứng từ xuất nhập khẩu / logistics.

## ✨ Tính năng
- 📊 Trích xuất dữ liệu từ **Excel, PDF, Ảnh (OCR), Word**
- 🌐 OCR hỗ trợ: Tiếng Việt, English, 中文简体, 中文繁體
- 🔍 So sánh tự động với highlight **Khớp / Sai lệch / Thiếu**
- 📦 So sánh theo lô hàng hoặc tự do
- 📋 Hỗ trợ 6 loại chứng từ: Tờ khai, Booking, B/L, Invoice, Packing List, Arrival Notice
- 📥 Xuất kết quả ra Excel/CSV

## 🚀 Cài đặt & Chạy

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔒 Bảo mật
Toàn bộ xử lý diễn ra trên server, không gửi dữ liệu ra bên ngoài.
